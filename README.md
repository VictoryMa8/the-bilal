# The Bilal

The Bilal is for those seeking to learn more about the current world and curate the history that shapes current perspectives. Friendly debate and the curation of information are both encouraged. The Bilal provides a space for those willing to learn new perspectives.

## Main Features

- Choose from 4 curated news topics to explore and debate on
- Vote for the following month's topics for debate
- Create new threads or respond to existing threads on one of the current topics
- In threads, link to articles that are related to the topic at hand
- Leave comments on these articles to curate historicity and authenticity
- As a journalist, create and maintain these articles and accept suggestions by users
- Shape the world with real, human-made interaction and discussion for a better
